# Chat With Private Data

## Environment

- Python 3.10

[streamlit docs](https://docs.streamlit.io/)

## Install

```
pip install -r requirements.txt
```

## To run the app

### Locally

```shell
streamlit run app.py
```

http://localhost:8080

### On Cloud9

```shell
streamlit run app.py --server.port <port>
```
